¿Por qué el sitio está inactivo?

Debido a trabajos de mantenimiento programados, nuestro sitio web se encuentra temporalmente inactivo.

¿Qué es el tiempo de inactividad?

Esto se refiere al tiempo durante el cual una computadora o un sistema informático no está funcionando debido a problemas técnicos, actualizaciones de software, mantenimiento o fallas de hardware.
